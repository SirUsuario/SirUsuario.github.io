Dead Pixeled Runner

Dead Pixeled Runner was my first personal project and my first 2D game! It is a simple platformer runner with multiple tints of personalization.
Enemies, instead of defeating you, increase the difficult level adding gameplay changes, but it is possible to avoid them buying upgrades at the end of every match. One of my best development experiences, using my engineer skills to improve the design of the game.
Art by Mhamed Bencrimo.

Antimatter

Antimatter is a 3D arcade shooter with bullet hell elements, where we will fight against waves after waves of different enemies.
The game was inspired by classic Ikaruga and displays a score system based on streaks and ammunition handling. Antimatter was not my first 3D game, but it was the first one completed all by myself. 3D models by Francisco Pacheco Gómez. 2D by Francisco Javier Podadera.

Down the Robin Hole

Down the Robin Hole was an approach to the graphic adventure genre, with a heavy narrative focus. Following Emmanuel, an ordinary man figthing against burocracy in order to retrieve an inheritance.
This project was influenced by Asterix and the twelve trials, and is my most ambicious personal project to date, including more than one hundred pages of dialogue, multiple dialogue branches and different endings determined by our choices.
Created using an 3D models library.

Descubre Málaga: Escape Game

Descubre Málaga: Escape Game es juego de Scape Room para teléfonos móviles realizado dentro de la empresa de gamificación WIIMO. El juego es una experiencia interactiva en la que el jugador deberá resolver distintos puzles a los cuales deberá acceder desplazándose físicamente a algunas localizaciones dentro de un mapa real.
Mi trabajo consistió en el diseño de alguno de los puzles y la creación del nivel de sigilo del final del juego.

Descubre Málaga: Escape Game is a scape room game made for android devices alongside WIIMO gamification bussiness. It is an interactive experience where players must travel to different places to unlock its levels, using a real map to track the locations.
I acted as a designer for some of the puzzles and the stealth mission level at the end.

Adventure Street Halloween

Adventure Street Halloween es juego de terror para teléfonos móviles realizado dentro de la empresa de gamificación WIIMO. El juego es una experiencia interactiva en la que el jugador deberá resolver distintos puzles a los cuales deberá acceder desplazándose físicamente a algunas localizaciones dentro de un mapa real. El juego está pensado para ser adaptado a distintas ciudades, favoreciendo la visita de monumentos característicos de cada localización. Para implementar el mapa y las funciones de geolocalización se hizo uso del paquete de Mapbox para Unity.

Adventure Street Halloween is a horror puzzle game made for android devices alongside WIIMO gamification bussiness. It is an interactive experience where players must travel to different places to unlock its levels, using a real map to track the locations.
It is designed to adapt itself to different cities, placing its levels on important landmarks. The map functionality was implemented using the Mapbox package for Unity.

Chronos Builder

Chronos Builder is a software tool for creating tabletop role-playing game maps. The tool allows you to generate maps procedurally and export them into images for printing or files you can import into Foundry and other VTTs. Chronos was product of an successful Kickstarter campaign, and currently counts with the DLC Wishing Well, which includes new assets and features. It was developed with Unity3D.
